# Python爱鲜蜂
## 一、环境说明
- python3.5

## 二、代码获取
```
# SSH方式
git clone git@github.com:iphone3/Python1807AXF.git

# HTTP方式
git clone https://github.com/iphone3/Python1807AXF.git
```

## 三、安装项目依赖
```
# 切换到项目目录中
pip install -r requirements.txt
```
> pip是Python3的版本!

## 四、启动项目
```
python manage.py runserver

# 浏览器中输入 【移动端项目，所以显示是小屏幕】
127.0.0.1:8000/axf/
```
